# Dagan's Personal Site

Site hosted at <https://dagans.dev>

## Hosting locally

```bash
$ git clone --recursive git@github.com:Property404/personal-site
...
$ cd personal-site
...
$ ./setup.bash
...
$ php -S 127.0.0.1:8000
PHP 8.0.16 Development Server (http://127.0.0.1:8000) started
```
